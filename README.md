
# 2026Gallop - 春节祝福生成器
一个庆祝2026农历马年的交互式新年祝福H5页面。寓意新年奔腾向前 (Gallop)，包含动态视觉效果与祝福互动。

一个精美的春节祝福在线生成工具，帮助您快速创建个性化的新春祝福卡片。

## ✨ 功能特点

- **头像选择**：提供多种春节主题头像供选择
- **祝福语库**：精选多种新年祝福语，一键选用
- **场景切换**：多种新春场景背景，随心切换
- **实时预览**：生成效果实时可见
- **二维码分享**：生成专属祝福二维码，方便分享

## 🚀 使用方法

1. 在「个性化设置」区域选择心仪的头像
2. 选择一句祝福语
3. 挑选喜欢的场景背景
4. 预览祝福效果
5. 点击分享，生成二维码发送给亲朋好友

## 🛠️ 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- 原生 JavaScript

## 📦 安装部署

### 本地运行

```bash
# 克隆项目
git clone https://github.com/GitFn/2026Gallop.git

# 或克隆项目
git clone https://gitee.com/GitFn/2026Gallop.git

# 进入项目目录
cd 2026Gallop

# 直接用浏览器打开 index.html
# 或使用本地服务器
npx serve .
```

### 在线部署

本项目可直接部署到任何静态网页托管平台：

- Netlify
- GitHub Pages
